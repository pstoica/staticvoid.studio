What We Collect

This app uses PostHog analytics to understand how features are being used. We collect:

Anonymous Usage Data

  • Feature interactions: Which features you use (e.g., "planet created", "composition saved")
  • App events: App launches, settings changes, feature usage patterns
  • Device information: Device type (iPhone/iPad), OS version, app version
  • Session data: How long you use the app, which screens you visit

What We Don't Collect

  • No personal information (name, email, phone)
  • No location data
  • No content you create (compositions, saved files, planet data)
  • No user identifiers (uses anonymous device ID only)
  • No IP addresses
  • No audio recordings
  • No MIDI data

Why We Collect It

To improve the app by understanding:

  • Which features are most/least used
  • Where users might be getting stuck
  • What crashes or errors occur
  • Which devices/OS versions need testing

Your Privacy Controls

Opt-in Required

  • Analytics are disabled by default
  • You're asked for consent on first launch
  • You can decline without any impact on app functionality

Change Anytime

  • Go to Settings → Privacy to enable/disable analytics
  • Changes take effect immediately
  • Past data cannot be deleted (it's already anonymous)

Technical Details

PostHog Configuration

  • Host: us.i.posthog.com (US-based servers)
  • IP Anonymization: Enabled (GDPR/CCPA compliant)
  • Feature Flags: Disabled (no auto-capture)
  • Session Recording: Disabled (never enabled)
  • User Identification: Anonymous device UUID only

Data Retention

  • PostHog retains data according to their retention policy
  • Typical retention: 7 years for event data
  • No way to delete individual user data (it's already anonymous)

Third-Party Sharing

  • Data is sent to PostHog only (not shared with other services)
  • PostHog may use subprocessors (AWS, etc.) — see their privacy policy

Compliance

GDPR (Europe)

  • Explicit consent required before data collection
  • IP anonymization enabled
  • Right to disable (no deletion needed — data is anonymous)
  • Transparent about what's collected

CCPA (California)

  • Opt-in required (not just opt-out)
  • No "sale" of personal information (data is anonymous)
  • Clear disclosure of collection practices

Apple Requirements

  • App Store Privacy Nutrition Label: "Product Interaction" + "Identifiers" (Device ID)
  • No IDFA or ad tracking
  • Privacy policy link required in App Store Connect
